The code for my teams capstone project named DABAM. Named after our groups intial DABAM is an automated nanoparticle synthesizer which aims to consistently produce nanoparticles. 

**Problem Statement**
Nanoparticle syntesis requires every fluid to have specific volumes dispensed at specific flow rates with varrying time rates between the reactants. 
Currently the QNC labs at the University of Waterloo synthesis nanoparticles manually by using syringes and electronic pipettes, this way of production does not gaurantee flow rates and time.
This method is quite inconsistent and results in a lot of waste products. Our machine aims to automate this process to increase consistency and free up researcher labour so that they may focus on groundbreaking research.

**System Composition**
The system utilizes the following components
1. Mechanical - Stepper motors, lead screws, e-stop, 2.8" TFT touchscreen
2. Sensors - limit switches, hall effect sensors
3. Software/Microcontroller - STM32 IDE
